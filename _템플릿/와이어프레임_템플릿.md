# ⚠️ 폐기된 템플릿 — 와이어프레임(.md)

> 상태: `🔴 폐기` | 대체: 기획화면 HTML

이 템플릿(텍스트 기반 와이어프레임 `.md`)은 **더 이상 사용하지 않습니다.**
화면 단위 산출물은 **기획화면 HTML** 로 일원화되었습니다.

## 새 화면을 만들 때

1. 해당 프로젝트 `02_기획화면/` 폴더에서 **가장 최근에 수정된 `SCR-*.html`** 파일을 기준 파일로 읽습니다.
2. 화면 ID·화면명·내용만 교체하여 새 `SCR-{화면ID}_{화면명}.html` 로 저장합니다.
3. 반드시 유지할 구조 (자세한 규칙은 `CLAUDE.md` → "기획화면 HTML 파일 생성 규칙"):
   - 문서 목차: 1. 서비스 개요 / 2. 화면 목록 (IA) / 3. 화면 기능 명세 / 4. 데이터 명세 / 5. 부록
   - CSS 디자인 토큰(`:root`)·공통 스타일은 수정하지 않음
   - 노트 저장 키(`NOTES_STORAGE_KEY_xxx` 변수·값)를 화면 ID에 맞게 변경

> 화면 ID·기능 ID에는 프로젝트 코드를 포함합니다. 예: `SCR-CBAM-001`, `F-CBAM-01`.
