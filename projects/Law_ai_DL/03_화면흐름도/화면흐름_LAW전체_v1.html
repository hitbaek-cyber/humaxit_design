1. 개요

대상 기능법률자문 상세 조회 — 코멘트 추가 / AI 코멘트 자동 생성
진입점법률자문 요청 등록 또는 법률자문 조회 목록에서 건 클릭
종료점코멘트 저장 완료 / 상세 조회 화면 닫기
관련 사용자자문 요청자, 법무 담당자

2. 전체 흐름

flowchart TD
    A([법률자문 메뉴 진입]) --> B{진입 경로}
    B -->|요청 등록 후 저장| C[법률자문 상세 조회]
    B -->|조회 목록에서 건 클릭| C

    C --> D[자문 요청 내용 + 코멘트 영역\n동시 노출]
    D --> E{코멘트 추가 클릭}
    E --> F[코멘트 작성 패널 오픈\n자문 내용 가림 없이 동시 열람]
    F --> G{AI 코멘트 생성 버튼 클릭?}
    G -->|예| H[AI가 요청 제목/본문/첨부/\n이전 코멘트 이력/수신자 정보 기반 초안 생성]
    H --> I[리치텍스트 에디터에\n생성된 초안 자동 입력]
    I --> J[사용자가 초안 검토·수정]
    G -->|아니오| J
    J --> K{저장}
    K -->|저장| L[코멘트 등록 완료\n상세 화면 갱신]
    K -->|취소| D
    L --> M([종료 또는 다른 자문 건 이동])

flowchart TD
    S1([SCR-LAW-001 진입]) --> S2[자문 요청 내용 영역 표시\n제목/본문/첨부/수신자]

    S7 -->|실패/오류| S9[오류 안내 + 수동 작성 유도]
    S8 --> S10[사용자 검토·수정]
    S9 --> S10
    S5 -->|아니오| S10
    S10 --> S11{저장 확인}
