## 논의 내용

### 안건 1 — CBAM 모듈: 고객 관점 기능 추가

**논의 요약**
> 고객 관점에서 필요한 기능을 추가하는 방향 검토.

**주요 의견**
- CN코드 별 벤치마크 및 인증서 구매 수량/금액 조회 기능 추가 필요.
- 기지불탄소가격에 대한 명확한 규제는 현재 없음. 현재 확인 가능한 수준까지 적용.

**CN코드 별 벤치마크 & CBAM 인증서 가격 인터페이스 조사 결과** (노준호, 2026.05.18 추가)
- **EU 집행위원회 API**: 현재 제공하는 API 기능은 없음.
- **CN코드별 벤치마크 값**: 기준값으로 자주 변경되는 값이 아니므로 엑셀을 등록하여 관리하더라도 충분함.
- **CBAM 인증서 가격**: 한재 분기별로 특정 사이트에서 제공. '27년부터 주 단위로 제공하게 되면 CBAM 등록부(Registry)에서 API 제공할 수도 있다고 함.

**결론**
> ✅ CN코드별 벤치마크 값은 엑셀 등록 관리 방식 채택. CBAM 인증서 가격도 분기별 수동 등록으로 관리. 당장 EU API 인터페이스는 고려하지 않아도 됨. → [D-001](../결정로그.md)

### 안건 2 — CBAM 모듈: 기지불탄소가격 적용 방향

**논의 요약**
> '26년 5월 제 3국 기지불탄소가격에 대한 이행 법안 Release 예정. 법안 확정은 2026년 하반기 중 예정.

**주요 내용**
- K-ETS 감우만 하더라도 유상, 무상, 상쇄 등 배출권 처리 방식이 다르기 때문에 이행 법안 확정되면 K-ETS 배출권에 대한 이해도를 높이면서 기지불탄소가격 기능도 구현되어야 함.
- **제 3국 기지불탄소가격 유효 탄소가격 산정 조건**: 원산지국 정부로부터 매해 제공받은 무상할당 혜택, 재정적 보조금(Subsidies), 세금 리베이트(Rebates) 및 직간접적인 면세 혜택 금액을 모두 제외한 순수 'Net 지불 비용'만을 유효 탄소가격으로 인정. 탄소 배출량(Emissions) 자체를 직접 깎아주는 구조가 아니라 수입업자가 최종 제출해야 하는 CBAM 인증서 수량(비용 단위)을 차감해 주는 통화적 비용 정산 방식.
- **노준호**: 지금 수준으로 이해하기로는 국내 무상 할당량 이런 거 인정하지 않고, 국내 배출권 가격 기준으로 CBAM 인증서를 감면하여 주는 것 같음. 여하튼 초안 기준으로 2026년 하반기에 최종 법안 확정 및 채택 시에 Deep Dive 필요함.

**감면 인증서 수량 산식**
> 감면 인증서 수량 = (원단위 내재 배출량 × 유효 탄소가격) / 유럽집행위원회 고시 연평균 CBAM 인증서 기준가격 × 수입 물량(톤)

**결론**
> ✅ 기지불탄소가격은 우선 0으로 가정하여 계산. 이행 법안 확정(2026년 하반기 예정) 시 Deep Dive 진행. → [D-002](../결정로그.md)

### 안건 4 — 영업: 핵심기능 — 내재배출량 기준 CBAM 인증서 비용 가시성 제공

**논의 요약**
> 고객 관점 최우선 개발 필요 항목으로 내재배출량 산정 기능 기반 CBAM 인증서 구매 비용 가시성 제공 기능이 선정됨.

**제안된 기능 흐름 (결론)**
> 1) 제품의 고유 내재 배출량을 산정하고
> 2) 데이터의 실제/기본값에 따른 Benchmark 기준값을 갖고
> 3) 기지불탄소가격은 우선 0으로 가정하고
> 4) 분기별 CBAM 인증서 가격으로 CBAM 인증서 구매 수량을 계산하는 기능 제공

**결론**
> ✅ 위 4단계 흐름으로 CBAM 인증서 구매 수량 계산 기능 구현 방향 확정. → [D-003](../결정로그.md)
