## 화면 구성 요약

### SCR-CBAM-004-L · 목록 화면 (Type A)
- **필터**: 날짜(최근수정일 기준, D=null), 상태(전체/사용/사용안함), 검색(제품코드·제품명)
- **그리드 컬럼**: 제품코드 / 제품명(링크→상세) / CN 코드 / 전구체 여부 / 상태 / 수정자 / 최근수정일
- **기본 정렬**: 최근수정일 내림차순. 전 컬럼 정렬 기능 제공
- **테이블 표시 선택**: 전체(D) · 자재코드/CN코드/전구체여부/상태/수정자/최근수정일
- **[+등록]**: 등록 화면으로 이동

### SCR-CBAM-004-R · 등록 화면 (Type B-1)
| 필드 | 필수 | 비고 |
|------|------|------|
| 제품 | Y | 자재 마스터 검색 드롭다운. 자재코드·자재명 표시 |
| 구분 | Y | 라디오. D=보고대상 제품 / 전구체 |
| CN 코드 | Y | 직접 입력 불가. [CN 코드 조회] 팝업 선택 |
| HS 코드 | N | 최대 12자리 숫자 |
| 생산 경로 | Y | 최대 9개. 품목별 옵션 상이 |
| 매개변수 | N | 배출량 산정 입력 변수. [추가]로 행 추가 |
| 설명 | N | 자유 텍스트 |

### CN 코드 조회 팝업
- 필터: 구분(전체/시멘트/전기/수소/철강/비료/알루미늄), CN 코드 검색
- 테이블: 선택(라디오) / 구분 / 품목군 / 품목(한글+영문) / CN 코드
- [확인]: 선택 코드 → CN 코드 필드에 반영
