배경 및 목적

EU CBAM(탄소국경조정메커니즘)은 철강·시멘트·알루미늄·비료·전력·수소 등 탄소집약적 상품 수입 시 EU ETS 탄소가격에 상응하는 비용을 부과하는 제도입니다. 수출 기업은 제품별 CN(Combined Nomenclature) 코드에 매핑된 벤치마크(기준 탄소배출량)와 무상할당 인수(Free Allocation Factor)를 기반으로 CBAM 인증서 구매 수량을 산정해야 합니다.

본 화면은 EU 집행위가 공표한 CN코드별 벤치마크·무상할당 기준값을 조회·관리하는 마스터 데이터 화면입니다.

무상할당 산정 공식
CBAM 인증서 구매 수량 = 내재 배출량(tCO₂e) × (1 − 무상할당 인수) − 원산지 탄소가격 상당분
무상할당 인수는 2026년부터 2034년까지 단계적 감소 예정 (Phase-out 스케줄 적용)

배경 및 목적

EU CBAM 인증서 구매 수량 산정의 핵심은 제품별 출하량(수출량)입니다. 본 화면은 생산 공장별·기간별 출하 데이터를 수집·등록·관리하는 화면으로, 직접 입력 / 파일 업로드 / ERP 연동의 3가지 방식을 지원합니다.

등록된 출하량은 내재배출량·인증서 비용 산정의 입력값으로 자동 연계됩니다.

배경 및 목적

EU CBAM(탄소국경조정메커니즘) 규정에 따라 수출 기업은 수출 제품의 내재 배출량에 상응하는 CBAM 인증서를 구매해야 합니다. 본 화면은 연간 CBAM 인증서 구매 예상 비용을 분기별로 산출·조회하는 화면입니다.

산출 결과는 입력된 출하량·배출량 데이터를 기반으로 한 예상 수치이며, 실제 인증서 구매 시점의 가격 및 규정에 따라 변동될 수 있습니다.

영역설명
필터기간(분기) + 생산공장 드롭다운

예상 수량·가격 카드필터 조건 기준 분기별 필요 인증서 수량 및 유로 환산 금액
데이터 테이블제품별 출하량·배출량·필요 인증서 수량·발생 비용 상세
출하량 현황 모달출하량 클릭 시 제품별 월별 수출 내역 팝업

화면 기능 명세 — SCR-CBAM-003

F-01   필터 조회

  • 기간(분기) 및 생산공장 드롭다운 필터 제공
  • 필터 변경 즉시 인증서 가격 카드·수량 카드·테이블 동시 갱신
  • 기본값: 기간 전체, 생산공장 전체

F-02   인증서 가격 카드

  • EU ETS 시장가격 기반 분기별 인증서 가격 표시
  • 확정 분기: 금액(€) 파란색 강조 표시
  • 미확정 분기: "발표 예정" 회색 텍스트
  • 가격 데이터는 관리자 화면에서 수동 입력 (SCR-CBAM-010 연계, 미작성)

F-03   예상 수량·가격 카드

  • 분기별 필요 인증서 수량(개) 및 유로 환산 금액 표시
  • 인증서 가격이 미확정인 분기는 "-" 표시
  • 합계 박스: 전체 분기 합산 수량 및 금액
  • ? 아이콘 호버 시 툴팁: "적용 환율: 유럽중앙은행(ECB) 전 영업일 기준 고시 환율"

F-04   데이터 테이블

  • 컬럼: 생산공장 / 제품 / 출하량(t) / 고유내재배출량(tCO2e) / 무상할당량(tCO2e) / 필요 인증서 수량 / 기지불 탄소가격(₩) / 발생 비용(€)
  • 필요 인증서 수량이 0인 경우 파란색으로 강조 표시 (구매 불필요)
  • 제품명 클릭 → 제품 상세 화면 이동 (SCR-CBAM-002, 미작성)
  • 출하량 클릭 → F-05 출하량 현황 모달 팝업

F-05   출하량 현황 모달

  • 제품코드/제품명, 생산공장 정보 표시
  • 연도 드롭다운으로 연도별 조회 전환
  • 월별 출하량(t), 수정일, 수정자 표시
  • 출하량 합계 자동 산출하여 하단 표시
  • 모달 외부 클릭 또는 ✕ 버튼으로 닫기

비즈니스 규칙

BR-01   필요 인증서 수량 산정 공식

필요 인증서 수량 = MAX(0, (고유내재배출량 − 무상할당량) × 출하량)

발생 비용(€) = 필요 인증서 수량 × 분기 인증서 가격 − (기지불 탄소가격 ÷ ECB 환율)

BR-02   환율 적용 기준

  • 적용 환율: 유럽중앙은행(ECB) 전 영업일 기준 고시 환율
  • 환율은 매 영업일 자동 업데이트 (ECB API 연동)
  • 당일 환율 미수신 시 전전 영업일 환율 적용 후 알림 발송

BR-03   무상할당 Phase-out

  • 무상할당 인수는 2026년부터 단계적 감소 (2034년 0%)
  • 연도별 무상할당 인수는 EU 집행위 공표값 기준 시스템 반영

BR-04   인증서 가격 미확정 처리

  • 인증서 가격이 미확정(PENDING)인 분기는 수량 표시, 금액은 "-" 처리
  • 가격 확정 즉시 자동 금액 재산출 및 화면 갱신
