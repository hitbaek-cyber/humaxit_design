진입 경로GNB > CBAM > 데이터 관리 > 출하 데이터 수집/관리
접근 권한관리자 / 일반 사용자

배경 및 목적

EU CBAM 인증서 구매 수량 산정의 핵심은 제품별 출하량(수출량)입니다. 본 화면은 생산 공장별·기간별 출하 데이터를 수집·등록·관리하는 화면으로, 직접 입력 / 파일 업로드 / ERP 연동의 3가지 방식을 지원합니다.

등록된 출하량은 내재배출량·인증서 비용 산정의 입력값으로 자동 연계됩니다.

화면 기능 명세

3.1 목록 화면 (SCR-CBAM-002-L)

Zone컴포넌트설명
Zone-2페이지 타이틀 + [+ 등록] 버튼등록 화면 이동
Zone-3날짜 필터 / 상태 드롭다운 / 공장 드롭다운복합 AND 필터
Zone-4결과 건수 / 테이블 표시 선택 / Excel 다운로드현재 조건 기준 CSV
Zone-5데이터 그리드공장코드·공장명·출하기간·제품 수·출하량·등록방식·상태
Zone-6페이지네이션50건 기본. 우클릭 컨텍스트 메뉴 (조회/수정/삭제)

3.2 등록 화면 (SCR-CBAM-002-R)

F-01 입력 방식 선택

  • 직접 입력: 그리드에서 제품 검색 후 출하량 수기 입력
  • 파일 업로드: 양식 엑셀 다운로드 후 업로드 (xlsx, xls, csv)
  • ERP 연동: 연동된 ERP 시스템에서 기간 지정 후 자동 가져오기

F-02 유효성 검사

  • 공장 + 출하 기간 조합 중복 불가 (등록 시 즉시 체크)
  • 출하량 0 이하 입력 불가
  • 필수 항목 미입력 시 인라인 오류 표시

3.3 조회/수정 화면 공통

  • 조회: 출하량 정보 테이블 (제품별 출하량 목록, 합계 행 고정)
  • 수정: 공장·출하 기간 변경 불가 (읽기 전용), 제품별 출하량만 수정 가능
  • 삭제: 수정 화면 [삭제] 버튼 → 확인 모달 → 삭제 처리
