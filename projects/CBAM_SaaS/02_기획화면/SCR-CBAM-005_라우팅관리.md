## 배경 (SAP PP 라우팅 참고)

SAP PP(생산계획) 모듈의 **라우팅(Routing / Task List)** 은 자재를 생산하는 **공정(Operation)의 순서**와 각 공정의
**작업장(Work Center)·제어키(Control Key)·표준시간(준비/기계/노무)** 을 정의하는 마스터 레코드다.
CBAM 산정에서는 제품 생산 공정별 활동·에너지 사용이 내재배출량 산정의 기초가 되므로, 라우팅을
기준정보로 관리해 BOM(자재구성) ↔ 라우팅(공정순서)을 함께 다룰 수 있도록 한다.

**SAP PP 핵심 개념 매핑**

| SAP PP | 본 화면 항목 |
|--------|-------------|
| Group / Group Counter | 라우팅 그룹 / 그룹 카운터 |
| Material / Plant | 제품(자재) / 플랜트 |
| Usage | 사용범위 (생산/하도급 등) |
| Overall Status | 상태 (작성중/릴리스됨) |
| Lot Size From~To | 로트크기 범위 |
| Valid From | 유효 시작일 |
| Operation (0010, 0020…) | 공정 번호 |
| Work Center | 작업장 |
| Control Key (PP01 등) | 제어키 |
| Standard Values (Setup/Machine/Labor) | 준비시간 / 기계시간 / 노무시간 |
| Component Allocation | 구성품(BOM) 할당 |
