## 1. 개요

### 1.1 배경 및 목적

EU CBAM(탄소국경조정메커니즘, Carbon Border Adjustment Mechanism)은 철강·시멘트·알루미늄·비료·전력·수소 등 탄소집약적 상품을 EU로 수입할 때 EU ETS(배출권거래제) 탄소가격에 상응하는 비용을 부과하는 제도입니다. 2026년 본격 시행을 앞두고 한국 수출 기업들은 다음 의무를 이행해야 합니다:

1. 제품별 CN(Combined Nomenclature) 코드 확인 및 벤치마크 조회
2. 공장별·기간별 출하량(수출량) 정확한 집계
3. 고유내재배출량과 무상할당량 기반의 CBAM 인증서 구매 수량 산정
4. 분기별 인증서 구매 비용 예측 및 예산 수립

**Compli·Law CBAM SaaS**는 이 복잡한 CBAM 대응 프로세스를 자동화하고 표준화하는 SaaS 플랫폼입니다.

### 1.2 목표

| 구분 | 목표 |
|------|------|
| **정량** | CN코드 검색 소요시간 30초 이내, 출하 데이터 등록 오류율 1% 미만, 엑셀 업로드 처리 30초 이내(최대 10,000행) |
| **정성** | EU 관보(Official Journal) 공표값 100% 반영, 분기별 CBAM 인증서 비용 예측을 통한 선제적 예산 수립 지원 |

### 1.3 범위 (Scope)

| 구분 | 내용 |
|------|------|
| **In Scope** | CN코드별 벤치마크·무상할당 기준값 조회/관리, 공장별·기간별 출하량 수집(직접 입력/파일 업로드), 분기별 CBAM 인증서 구매 예상 비용 산출·조회 |
| **Out of Scope** | ERP 인터페이스 자동 연동 (UI 숨김, 추후 오픈), CBAM 신고서 자동 제출, EU ETS 인증서 실거래, 탄소 크레딧 거래 |

> ⚠️ **ERP 연동 기능**은 개발 완료 후 별도 오픈 예정입니다.

### 3.2 기능 상세

#### F-CBAM-01: CBAM 벤치마크 조회

- **설명**: EU 집행위가 공표한 CN코드별 벤치마크(기준 탄소배출량)와 무상할당 인수(Free Allocation Factor) 조회
- **입력값**: CN코드 검색어, 섹터 필터, 상품 그룹 필터, 연도 필터
- **출력/결과**: CN코드 / 섹터 / 상품그룹 / 제품명 / 벤치마크값 / 무상할당 인수 / 적용 연도 / 상태
- **예외 처리**: 검색 결과 없을 경우 빈 상태(empty state) 표시
- **관련 화면**: [SCR-CBAM-001](../02_기획화면/SCR-CBAM-001_벤치마크.html)

#### F-CBAM-02: 벤치마크 마스터 관리

- **설명**: 관리자가 EU 관보 신규 공표 시 기준값을 등록·수정하고, 파일 일괄 업로드로 신규 연도분 데이터를 반영
- **입력값**: CN코드, 섹터, 상품그룹, 제품명, 벤치마크값, 무상할당 인수, 적용 연도
- **출력/결과**: 마스터 데이터 저장 및 목록 반영
- **예외 처리**: 동일 CN코드+연도 중복 등록 시 수정 화면으로 유도
- **관련 화면**: [SCR-CBAM-001](../02_기획화면/SCR-CBAM-001_벤치마크.html)

#### F-CBAM-03: 출하량 직접 입력

- **설명**: 생산 공장 선택 후 신고 기간을 지정하고, 제품별 월별 출하량(t)을 직접 입력
- **입력값**: 생산공장, 신고연도, 제품코드, 월별 출하량
- **출력/결과**: 출하 데이터 저장, 인증서 비용 산출 자동 반영
- **예외 처리**: 동일 공장+제품+기간 중복 등록 시 기존 데이터 수정 유도, 음수·비숫자 입력 차단
- **관련 화면**: [SCR-CBAM-002](../02_기획화면/SCR-CBAM-002_출하데이터수집관리.html)

#### F-CBAM-04: 출하량 파일 업로드

- **설명**: 정해진 엑셀 양식으로 공장별 출하 데이터를 일괄 업로드, 검증 후 미리보기 확인
- **입력값**: 엑셀 파일 (.xlsx), 생산공장 선택
- **출력/결과**: 파일 내용 검증 → 오류 안내 or 미리보기 → 최종 저장
- **예외 처리**: 필수 컬럼 누락, 존재하지 않는 공장/제품코드, 수량 오류 시 행별 오류 메시지 표시
- **관련 화면**: [SCR-CBAM-002](../02_기획화면/SCR-CBAM-002_출하데이터수집관리.html)

#### F-CBAM-07: 인증서 비용 조회

- **설명**: 등록된 출하량·배출량 데이터 기반으로 연간 분기별 CBAM 인증서 구매 예상 수량 및 비용 산출 (4단계 흐름: 내재배출량 → 벤치마크 적용 → 기지불 탄소가격 → 분기 인증서 가격, [D-003])
- **산출 공식**: `필요 인증서 수량 = MAX(0, (고유내재배출량 - 무상할당량) × 수출량)`, `발생 비용(€) = 필요 인증서 수량 × 분기 인증서 가격 - 기지불 탄소가격 환산액`
  - ⚠️ **1차 출시 한정**: 기지불 탄소가격 환산액은 **0으로 가정**한다([D-002]). 이행 법안 확정(2026 하반기 예정) 후 Deep Dive하여 반영.
- **입력값**: 기간(분기) 필터, 생산공장 필터
- **출력/결과**: 인증서 가격 카드(분기별), 예상 수량·비용 카드(분기별·합계), 제품별 상세 테이블
- **예외 처리**: 인증서 가격 미확정 분기는 "-" 표시, 필요 수량 0 이하는 0으로 표시(파란색 강조)
- **관련 화면**: [SCR-CBAM-003](../02_기획화면/SCR-CBAM-003_인증서비용조회.html)

#### F-CBAM-08: 수출량 상세 조회 (모달)

- **설명**: 인증서 비용 조회 테이블에서 수출량 클릭 시 제품별 월별 수출 내역 모달 팝업
- **입력값**: 제품코드 (클릭 트리거), 연도 드롭다운
- **출력/결과**: 제품코드·제품명, 생산공장, 월별 수출량(t), 수정일, 수정자, 합계
- **예외 처리**: 데이터 없는 월은 빈값 표시, 모달 외부 클릭 또는 ✕ 버튼으로 닫기
- **관련 화면**: [SCR-CBAM-003](../02_기획화면/SCR-CBAM-003_인증서비용조회.html)

- **제약사항**:
  - 벤치마크·인증서 가격 기준값은 EU 관보 공표 후 관리자가 **엑셀 등록 수동 관리** (EU API 자동 연동 미지원, 1차 출시) — [D-001]
  - ERP 연동 기능은 개발 완료 전까지 UI에서 숨김 처리
  - 인증서 가격은 미확정 분기의 경우 관리자가 추후 입력 (산출 결과 업데이트)
- **가정**:
  - 적용 환율은 유럽중앙은행(ECB) 전 영업일 기준 고시 환율 사용
  - CBAM 인증서 필요 수량이 0 이하인 경우 구매 불필요(0으로 처리)
  - **기지불 탄소가격은 1차 출시에서 0으로 가정**한다 — 제 3국 기지불 탄소가격 이행 법안 미확정. 법안 최종 확정(2026 하반기 예정) 후 Deep Dive하여 반영 — [D-002]
- **의존성**:
  - F-CBAM-07 인증서 비용 조회는 F-CBAM-03/04 출하량 데이터 및 F-CBAM-01 벤치마크 데이터가 선행 등록되어야 함

> **관련 결정**: [D-001](../04_히스토리/결정로그.md) 데이터 엑셀 등록 관리 · [D-002](../04_히스토리/결정로그.md) 기지불 탄소가격 0 가정 · [D-003](../04_히스토리/결정로그.md) 인증서 구매수량 4단계 계산 흐름
