배경 및 목적

EU CBAM(탄소국경조정메커니즘, Carbon Border Adjustment Mechanism)은 철강·시멘트·알루미늄·비료·전력·수소 등 탄소집약적 상품을 EU로 수입할 때 EU ETS(배출권거래제) 탄소가격에 상응하는 비용을 부과하는 제도입니다. 2026년 본격 시행을 앞두고 한국 수출 기업들은 다음 의무를 이행해야 합니다:

  1. 제품별 CN(Combined Nomenclature) 코드 확인 및 벤치마크 조회
  2. 공장별·기간별 출하량(수출량) 정확한 집계
  3. 고유내재배출량과 무상할당량 기반의 CBAM 인증서 구매 수량 산정
  4. 분기별 인증서 구매 비용 예측 및 예산 수립

Compli·Law CBAM SaaS는 이 복잡한 CBAM 대응 프로세스를 자동화하고 표준화하는 SaaS 플랫폼입니다.

목표

구분목표
정량CN코드 검색 소요시간 30초 이내, 출하 데이터 등록 오류율 1% 미만, 엑셀 업로드 처리 30초 이내(최대 10,000행)
정성EU 관보(Official Journal) 공표값 100% 반영, 분기별 CBAM 인증서 비용 예측을 통한 선제적 예산 수립 지원

범위 (Scope)

구분내용
In ScopeCN코드별 벤치마크·무상할당 기준값 조회/관리, 공장별·기간별 출하량 수집(직접 입력/파일 업로드), 분기별 CBAM 인증서 구매 예상 비용 산출·조회
Out of ScopeERP 인터페이스 자동 연동 (UI 숨김, 추후 오픈), CBAM 신고서 자동 제출, EU ETS 인증서 실거래, 탄소 크레딧 거래
⚠️ ERP 연동 기능은 개발 완료 후 별도 오픈 예정입니다. 현재 UI에서 숨김 처리됩니다.

제약사항 및 가정

제약사항
  • 벤치마크·인증서 가격 기준값은 EU 관보 공표 후 관리자가 엑셀 등록 수동 관리 (EU API 자동 연동 미지원, 1차 출시) — D-001
  • ERP 연동 기능은 개발 완료 전까지 UI에서 숨김 처리
  • 인증서 가격은 미확정 분기의 경우 관리자가 추후 입력 (산출 결과 업데이트)
가정
  • 적용 환율은 유럽중앙은행(ECB) 전 영업일 기준 고시 환율 사용
  • CBAM 인증서 필요 수량이 0 이하인 경우 구매 불필요 (0으로 처리)
  • 기지불 탄소가격은 1차 출시에서 0으로 가정한다 — 제 3국 기지불 탄소가격 이행 법안 미확정. 법안 최종 확정(2026 하반기 예정) 후 Deep Dive하여 반영 — D-002
의존성
  • F-CBAM-07 인증서 비용 조회는 F-CBAM-03/04 출하량 데이터 및 F-CBAM-01 벤치마크 데이터가 선행 등록되어야 함
관련 결정: D-001 데이터 엑셀 등록 관리 · D-002 기지불 탄소가격 0 가정 · D-003 인증서 구매수량 4단계 계산 흐름
